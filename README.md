# OnlySheet

## Описание проекта

OnlySheet - это система для безопасной работы с электронными таблицами на базе ONLYOFFICE Document Server. Проект создан для обеспечения защищенного доступа к табличным данным с учетом ролевой модели и ограничений на редактирование.

## Основные возможности

- **Ролевая модель доступа**:
  - Администратор: полный доступ к системе, управление пользователями и документами
  - Редактор: возможность редактировать разрешенные ячейки в документах
  - Читатель: только просмотр документов без возможности редактирования

- **Защита данных**:
  - Защита формул и важных ячеек от редактирования
  - Запрет на скачивание документов
  - Логирование всех действий пользователей

- **Совместная работа**:
  - Возможность одновременной работы нескольких пользователей
  - Отображение статуса и действий других пользователей в реальном времени

## Технический стек

- **Серверная часть**:
  - Node.js
  - Express.js
  - JWT для аутентификации

- **Клиентская часть**:
  - HTML/CSS/JavaScript
  - jQuery

- **Хранение данных**:
  - JSON файлы для пользователей и метаданных
  - Документы в файловой системе

- **Интеграция**:
  - ONLYOFFICE Document Server API

- **Контейнеризация**:
  - Docker
  - Docker Compose

## Установка и запуск

### Предварительные требования

- Docker и Docker Compose
- Node.js 14+ и npm

### Шаги по установке

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/t0uchY233/onlysheet.git
   cd onlysheet
   ```

2. Установить зависимости:
   ```bash
   npm install
   ```

3. Запустить ONLYOFFICE Document Server с помощью Docker Compose:
   ```bash
   cd onlyoffice
   docker-compose up -d
   ```

4. Создать администратора по умолчанию:
   ```bash
   node server/createAdminUser.js
   ```

5. Запустить приложение:
   ```bash
   npm start
   ```

6. Открыть приложение в браузере: http://localhost:3000

### Доступ к системе

После установки вы можете войти в систему с учетными данными администратора по умолчанию:

- Логин: `admin`
- Пароль: `admin`
